# 两数之和 Todo List

1. 两数之和

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target  的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

你可以按任意顺序返回答案。

## 1. 理论复习
- [ ] **理解题目要求**：
  - 确保完全理解题目要求和限制条件。
  - 理解输出是数组的索引，而不是数字本身。
  - 理解题目保证只有一个解。

- [ ] **复习暴力解法**：
  - 理解两层循环的逻辑。
  - 理解为什么内层循环从 `i + 1` 开始。
  - 理解时间复杂度 \(O(n^2)\) 和空间复杂度 \(O(1)\)。

- [ ] **复习哈希表解法**：
  - 理解哈希表的使用方法。
  - 理解如何通过一次遍历解决问题。
  - 理解时间复杂度 \(O(n)\) 和空间复杂度 \(O(n)\)。

## 2. 代码实现
### 暴力解法
```javascript
function twoSum(nums, target) {
    // 外层循环：遍历数组中的每个数字
    for (let i = 0; i < nums.length; i++) {
        // 内层循环：从当前数字的下一个数字开始遍历
        for (let j = i + 1; j < nums.length; j++) {
            // 检查当前数字 nums[i] 和另一个数字 nums[j] 的和是否等于目标值 target
            if (nums[i] + nums[j] === target) {
                // 如果找到满足条件的两个数字，返回它们的索引 [i, j]
                return [i, j];
            }
        }
    }

    // 如果没有找到答案，抛出错误（题目保证有解，所以这里不会执行）
    throw new Error("No two sum solution");
}

// 示例测试
console.log(twoSum([2, 7, 11, 15], 9)); // 输出：[0, 1]
console.log(twoSum([3, 2, 4], 6));      // 输出：[1, 2]
console.log(twoSum([3, 3], 6));         // 输出：[0, 1]
```

### 哈希表解法
```javascript
function twoSum(nums, target) {
    // 创建一个哈希表，用于存储已经遍历过的数字及其索引
    const map = new Map();

    // 遍历数组
    for (let i = 0; i < nums.length; i++) {
        // 计算当前数字与目标值的差值
        const complement = target - nums[i];

        // 检查哈希表中是否存在这个差值
        if (map.has(complement)) {
            // 如果存在，返回当前索引和哈希表中存储的索引
            return [map.get(complement), i];
        }

        // 如果不存在，将当前数字及其索引存入哈希表
        map.set(nums[i], i);
    }

    // 如果没有找到答案，抛出错误（题目保证有解，所以这里不会执行）
    throw new Error("No two sum solution");
}

// 示例测试
console.log(twoSum([2, 7, 11, 15], 9)); // 输出：[0, 1]
console.log(twoSum([3, 2, 4], 6));      // 输出：[1, 2]
console.log(twoSum([3, 3], 6));         // 输出：[0, 1]
```
